Basic Calculator




Calculate Maximum, Minimum, Mean, Median and Range.




This calculator was designed as part of a school project for my COP3813 class to incorporate html, javascript, css, and bootstrap

This calculator can be used to calculate the maximum, minimum, mean(average), median and range of three numbers.

To use, enter 3 numbers in the fields below, and click calculate

The form accepts numbers, including positive numbers, negative numbers, and decimals

The form only accepts numbers, rejecting any non numeric input using a simple input type="number"
